这是一个比较简陋的js自动观看视频脚本，水平比较低。

目前实现了自动播放视频并在视频结束后自动跳转到下一节。

但是由于不知道怎么在讨论那里写评论，所以还没添加自动在讨论那里写帖子的功能。

需要注意的是，由于还在摸索阶段，还没有绕开浏览器识别用户手势的检测门槛，所以打开油猴脚本后，第一节需要自己手动点击视频播放，让浏览器识别到用户手势，之后便能开始自动观看了。

有大佬也可以指导一下如何绕开用户手势， _(:з」∠)_

########################################################################

使用方法：

  1.在您的谷歌上下载油猴插件(tampermonkey)。
  
  2.在油猴右上角选项处新建脚本。
  <img width="418" height="545" alt="微信图片_20260709012850_239_155" src="https://github.com/user-attachments/assets/be7fc430-b372-4006-9784-e7a57bbdd34e" />

  3.复制仓库里的代码进去，保存。
  
  4.确保脚本开启，在心理教育（下）页面的第一节，手动点击视频播放，开始自动观看。

  #不要切换窗口或最小化会导致视频暂停，无法正常学习。
